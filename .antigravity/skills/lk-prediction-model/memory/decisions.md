# Skill Memory: Design Decisions

> Records architectural decisions made during implementation and WHY.
> Prevents the agent from re-debating settled decisions in future sessions.

## Decisions

*(Entries will be appended below as decisions are made)*

### Template

```
### [DECISION-NNN] Title
- **Date**: YYYY-MM-DD
- **Context**: What prompted this decision
- **Options Considered**: What alternatives were evaluated
- **Decision**: What was chosen
- **Rationale**: Why this option won
- **Consequences**: What this means for future work
```
