# Design An Interface

## Overview
Prioritizes "Premium Aesthetics" and "Visual Excellence" in UI design. We do not build MVPs; we build state-of-the-art interfaces.

## When to Use
- You are adding a new page or component to the AstroQ dashboard.
- You need a visual representation of complex astrological data (e.g., a 75-year timeline).

## The Design Principles
1. **Rich Aesthetics**: Vibrant colors, dark modes, glassmorphism, and dynamic animations.
2. **Modern Typography**: Use Google Fonts (Inter, Outfit, etc.).
3. **Interactive & Alive**: Hover effects and micro-animations.
4. **No Placeholders**: Use `generate_image` for realistic assets.

## The Process
1. **Moodboard**: Define the visual direction (e.g., "Cosmic Dark Mode with Neon Accents").
2. **Variant Generation**: Propose 2-3 design options.
3. **Mockup**: Create a high-fidelity visual using `generate_image`.
4. **Iterate**: Refine based on feedback until it feels "Premium."

## HARD-GATE
**Do NOT write frontend code until the visual design is approved.**
